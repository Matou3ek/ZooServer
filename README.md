## ZooServer Project
This project is used as Web Api for client application [Zoo](https://github.com/Matou3ek/Zoo). Web Api allows you to add, remove, update and list animal table.


## Requirements

Tomcat server
Mysql database
Java 1.7+


## Installation
copy ZooServer(zoo-1.0.0-BUILD-SNAPSHOT).war to <tomcat>/webapps folder
